# Dr. Soumya Pal - Academic Portfolio

This is the personal academic portfolio website for Dr. Soumya Pal, Environmental Economist and Policy Strategist at Krea University.

## Features

- **Responsive Design**: Adapts to all screen sizes.
- **Interactive Elements**: GSAP animations for staggered entry, parallax tilt effects, and morphing modals.
- **Modern Styling**: Custom CSS properties for easy theming.

## Getting Started

### Prerequisites

- Node.js installed on your machine.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/soumya-pal-portfolio.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd soumya-pal-portfolio
    ```
3.  Install dependencies (optional, but good for local dev server):
    ```bash
    npm install
    ```

### Running Locally

To start the local development server:

```bash
npx http-server
```

Open [http://localhost:8080](http://localhost:8080) in your browser.

## Deployment

This site is a static website (HTML/CSS/JS) and can be deployed to any static hosting service.

### Vercel (Recommended)

1.  Install Vercel CLI: `npm i -g vercel`
2.  Run `vercel` in the project directory.
3.  Follow the prompts to deploy to production.

### GitHub Pages

1.  Go to your repository settings on GitHub.
2.  Navigate to "Pages".
3.  Select the `main` branch as the source.
4.  Save.
